### 关于全局变量

```c
/* Step 1: Initialize basemem.
* (When use real computer, CMOS tells us how many kilobytes there are). */
basemem = 0x4000000;
maxpa = 0x4000000;
extmem = 0x0;
// Step 2: Calculate corresponding npage value.
npage = basenum >> 12;
```

后来看了一下在mmu.h里面有对SHIFT的定义，更严谨的来说应该是要写那个吧。



### 如何用命令行压缩文件

压缩服务器上当前目录的内容为xxx.zip文件

```
zip -r xxx.zip ./*
```

解压zip文件到当前目录

```
unzip filename.zip
```

这样就可以从实验环境copy代码了！
